# MultiFingerGestureX

Swipe or pinch with 3, 4, or 5 fingers anywhere on screen to perform system actions, launch apps, open custom panels, execute shell commands, and more.

## What It Does

<img width="480" height="340" alt="1" src="https://github.com/user-attachments/assets/0d1ceff6-58a7-4221-ab37-bb4ea93b5bad" />

Put three or more fingers on the screen:

| Gesture | Action |
|---------|--------|
| Short Tap | Back |
| Swipe Down | Screenshot |
| Swipe Up | Home (fast) / Recents (slow) |
| Swipe Left / Right | Switch to previous / next app |

三指或更多手指在屏幕上操作：

| 手势 | 效果 |
|------|------|
| 短按 | 返回 |
| 下滑 | 截屏 |
| 上滑 | 快速 → 回桌面 / 慢速 → 最近任务 |
| 左滑 / 右滑 | 切换上一个 / 下一个应用 |

## Requirements

- Android 15+ (API 35+)
- LSPosed or compatible Xposed framework
- Root access
- Scope: **System Framework** (`android`)
